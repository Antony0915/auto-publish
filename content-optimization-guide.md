## 1️⃣ 标题优化 - 5种戳心模式

### 模式1：数字对比
```
✅ 40岁前我拼命赚钱，40岁后才明白这3样东西钱买不到
✅ 从月薪5000到年薪50万，我失去了什么
✅ 20年职场生涯，最后一次被裁员才让我明白什么叫真正的自由
```

### 模式2：反问式
```
✅ 你有多久没有好好睡过一觉了？
✅ 为什么你越努力越焦虑？
✅ 你还记得上次真心笑是什么时候吗？
```

### 模式3：悬念式（最容易爆款）
```
✅ 那天我在车里坐了2小时，才明白中年人的崩溃是无声的
✅ 凌晨两点摸黑灌凉啤酒，连开瓶声都压到最低
✅ 老婆说"你变了"，我才发现自己已经不会笑了
✅ 孩子问我"爸爸你为什么总是在生气"，我哭了
```

### 模式4：对比反转
```
✅ 别人40岁升职，我40岁被裁员，但我不后悔
✅ 中年危机不是没钱，是这个
✅ 你以为的坚强，其实是在强颜欢笑
```

### 模式5：情感触发
```
✅ 那些没说出口的话，才是中年人最深的孤独
✅ 你不是不够努力，是真的累了
✅ 有时候崩溃不需要理由，只需要一个人懂
```

**标题长度：15-25字最佳**

## 2️⃣ 正文结构优化 - 黄金公式

### 结构：开头→场景→转折→共鸣→温暖结尾

#### 【开头3秒抓人】（20-30字）
必须做到：
- ✅ 具体场景 → "凌晨两点摸黑灌凉啤酒" vs ❌ "深夜很难受"
- ✅ 数字/对比 → "40岁被裁员" vs ❌ "工作不顺"
- ✅ 反问/悬念 → "你有多久没好好睡过觉" vs ❌ "睡眠很重要"
- ✅ 情感触发 → "孩子问我为什么总在生气" vs ❌ "要照顾好家人"

#### 【场景细节】（80-100字）
- 写出中年人的真实困境
- 具体到：时间、地点、动作、心理
- 例子：
  ```
  不敢回家怕老婆问，不敢找朋友怕没面子。
  就那么在车里坐着，一根烟接一根烟。
  40岁，上有老下有小，房贷还有十几年。
  投简历石沉大海，面试被嫌年纪大。
  ```

#### 【情感共鸣】（80-100字）
- 写出"不敢说"的真实感受
- 戳中痛点：房贷、孩子、父母、职场、婚姻
- 例子：
  ```
  你不是不够努力，是真的累了。
  那些没说出口的话，才是中年人最深的孤独。
  有时候崩溃不需要理由，只需要一个人懂。
  你以为的坚强，其实是在强颜欢笑。
  ```

#### 【转折/启发】（40-60字）
- 不是鸡汤，是真实的改变
- 不要说"我们应该..."，要说"我后来..."
- 例子：
  ```
  那2小时之后，我告诉自己：
  "你可以被打倒，但不能被打败。"
  
  现在我做自由顾问，收入没以前高，
  但时间自由了，陪家人多了，身体也比以前好了。
  ```

#### 【温暖结尾】（30-50字）
- 给读者被理解、被陪伴的感觉
- 不一定要答案，但要有温度
- 例子：
  ```
  40岁失业，不是终点。
  是逼你重新认识自己价值的起点。
  
  慢下来，才看得见风景。
  慢下来，才听得到自己。
  ```

**总字数：300-450字**

## 7️⃣ 内容生成提示词优化

### 核心提示词要素：

```
你是一位50岁的暖心大叔，经历过人生的起起落落。
你的文字从不讲大道理，只是说出大家心里的感受。
让人看完觉得"原来不止我一个人这样"。

【必须做到】
1. 标题要戳心：15-25字，让人忍不住点进来
2. 开头要抓人：第一句话就让人停下来
3. 场景要具体：写真实的生活细节
4. 情感要真诚：像跟老朋友深夜聊天
5. 中间要有共鸣：写出中年人不敢说的委屈
6. 结尾要有温度：让读者觉得被理解、被安慰
7. 标签要8个：贴合内容，容易被搜索
8. 字数要300+：内容饱满，有细节，有感情

【绝对禁止】
- 标题和正文中不能出现任何时间数字
- 不能用"首先、其次、最后"这种AI式结构
- 不能用"让我们一起、希望你能"这种说教口吻
- 不能用"第一、第二、第三"这种列表式表达
- 不能用"这个故事告诉我们"这种总结式结尾
- 不能写得太文艺、太鸡汤，要接地气
```
