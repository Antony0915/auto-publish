## 可用接口

### 1. 检查 VIP 状态
- 接口: GET /v1/team/vip/check
- 用途: 检查当前团队是否为 VIP

### 2. 账号数据查询
- 接口: GET /v1/account/overview-new
- 用途: 获取账号概览数据

### 3. 作品数据查询
- 接口: GET /v1/content/works
- 用途: 获取作品数据列表

### 4. 内容发布
- 接口: POST /v1/task/publish
- 用途: 创建发布任务

### 5. 获取地区列表（用于设置代理）
- 接口: GET /v1/region/tree
- 用途: 获取可用的地区代理列表
